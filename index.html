Mes projets

Projet 1

Développement d'une application web complète pour la gestion et le stockage des traces des tests réalisés sur des produits électroniques. Cette application, conçue avec Angular 17 pour le frontend et .NET 7 pour le backend, offre une interface utilisateur intuitive permettant de visualiser, rechercher et organiser les données des tests. Elle intègre également une base de données relationnelle (MySQL) pour assurer une gestion efficace et sécurisée des informations critiques, tout en respectant les exigences de performance et d'évolutivité.
